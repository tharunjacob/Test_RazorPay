# Quick Push to GitHub - Use Cursor's Built-in Git

## ✅ All files are ready! Just follow these 3 steps in Cursor:

### Step 1: Open Source Control (5 seconds)
- Press **`Ctrl+Shift+G`** 
- OR click the Source Control icon in the left sidebar (looks like a branch/fork icon)

### Step 2: Initialize & Stage (10 seconds)
- If you see "Initialize Repository" button → Click it
- Click the **"+"** button next to "Changes" to stage all files
- OR right-click "Changes" → "Stage All Changes"

### Step 3: Commit & Push (15 seconds)
- Type commit message: `Initial commit: TrackSpendz app ready for Vercel deployment`
- Press **`Ctrl+Enter`** to commit
- Click **"Publish Branch"** button (top of Source Control panel)
- When asked for remote URL, enter: `https://github.com/tharunjacob/Test_RazorPay.git`
- Click "OK" or "Publish"

**That's it!** Your code will be pushed to GitHub.

---

## If "Publish Branch" doesn't appear:

1. After committing, click the **"..."** (three dots) menu at the top
2. Select **"Push"** or **"Push to..."**
3. Enter remote: `origin`
4. Enter URL: `https://github.com/tharunjacob/Test_RazorPay.git`

---

## Need to add remote manually?

If you need to add the remote first:
1. Click the **"..."** menu
2. Select **"Remote"** → **"Add Remote"**
3. Name: `origin`
4. URL: `https://github.com/tharunjacob/Test_RazorPay.git`
